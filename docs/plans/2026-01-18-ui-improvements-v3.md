# UI 改进设计 V3

## 概述
第三轮 UI 改进，包含木鱼弹窗、日历、成就系统、SOS悬浮球的重设计。

## 1. 木鱼弹窗改造

### 背景样式
- 渐变背景：`#FFE5E5`（淡粉红）→ `#FFF5F5`（近白粉）
- 叠加玻璃模糊：`backdrop-filter: blur(20px)`
- 整体温暖柔和

### 音效
- 使用 Web Audio API
- 音效文件：`/public/sounds/wooden-fish.mp3`
- 允许快速连续敲击时音效叠加

### 视觉
- 保持棕色木鱼造型
- 「功德+1」浮动文字改为金黄色

## 2. 日历样式重设计

### 布局
- 7 列网格（周日到周六）
- 圆角方块：`border-radius: 8px`
- 方块间距 4px
- 自定义渲染，移除 react-day-picker 默认样式

### 颜色规则
- 无记录：浅灰 `#F5F5F5`
- 仅自律：绿色 `#34C759` + 白字
- 仅起飞：红色 `#FF3B30` + 白字
- 既自律又起飞：红色优先
- 今天：蓝色边框

### 头部
- 月份切换箭头在标题两侧
- 星期标题行灰色居中

## 3. 成就系统

### 等级设定（按起飞次数）
| 等级 | 称号 | 起飞次数门槛 |
|------|------|-------------|
| 1 | 凡人 | 0 |
| 2 | 鹿王 | 10 |
| 3 | 机组人员 | 30 |
| 4 | 机长 | 60 |
| 5 | 传奇机长 | 100 |

### 展示位置
- 个人页头像下方显示当前等级徽章
- 首页主圆环下方小字显示等级

## 4. SOS 悬浮球重设计

### 样式
- 火焰渐变：深红 `#DC2626` → 橙红 `#F97316`
- 发光效果：`box-shadow` 红色光晕
- 保持「实在忍不住了！」飘语动画

### 尺寸
- 保持 56x56px 圆形
- 中间显示「SOS」白色文字
