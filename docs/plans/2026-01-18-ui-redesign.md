# 鹿了么 UI 重设计方案

## 设计目标

将现有 UI 改造为真正的 iOS 17+ 液态玻璃风格，全中文界面。

## 设计决策

| 项目 | 决策 |
|------|------|
| 风格 | 液态玻璃 (Liquid Glass) |
| 背景 | 柔和蓝白渐变 |
| 色调 | 蓝白为主 |
| 按钮 | 胶囊形 iOS 原生风格 |

---

## 第1部分：视觉基础

### 背景层
整个应用使用固定的渐变背景，从顶部的淡蓝色 (`#E8F4FC`) 过渡到底部的纯白 (`#FFFFFF`)。

### 玻璃卡片
所有卡片使用统一的液态玻璃样式：
- 背景：`rgba(255, 255, 255, 0.65)` 半透明白
- 模糊：`backdrop-blur: 20px`
- 边框：`1px solid rgba(255, 255, 255, 0.5)` 微光边缘
- 阴影：`0 8px 32px rgba(0, 0, 0, 0.08)` 柔和投影
- 圆角：`24px` 大圆角

### 胶囊按钮
- 主按钮：蓝色填充 (`#007AFF`)，白色文字，全圆角
- 次按钮：白色毛玻璃背景，蓝色文字
- 危险按钮：红色填充 (`#FF3B30`)
- 点击效果：`scale(0.97)` 轻微缩小

---

## 第2部分：页面布局与组件

### 整体布局
- 顶部安全区：适配刘海屏，使用 `env(safe-area-inset-top)`
- 底部导航栏：固定在底部的毛玻璃导航条，4个 Tab（首页、社区、榜单、我的）
- 内容区域：可滚动，底部留出导航栏高度 + 安全区

### 首页核心组件

**1. 数据卡片组（顶部）**
两个并排的小型玻璃卡片：
- 左：「历史最高 XX 天」
- 右：「累计功德 XXX」

**2. 主圆环（中央焦点）**
大型圆形玻璃容器，居中显示：
- 上方小字：「已坚持」
- 中间超大数字：当前天数
- 下方小字：「天」

**3. 打卡按钮组**
两个并排胶囊按钮（仅当日未打卡时显示）：
- 绿色：「坚持戒律」
- 红色：「快乐起飞」

**4. 修行日历**
底部玻璃卡片内嵌日历，用颜色标记历史状态。

## 第3部分：其他页面与导航

### 底部导航栏
毛玻璃背景，4个 Tab：
- 「首页」- 房子图标
- 「社区」- 人群图标  
- 「榜单」- 奖杯图标
- 「我的」- 人像图标

选中状态：蓝色图标 + 文字，顶部有一条蓝色指示线

### 登录/注册页
- 顶部大标题「鹿了么」+ 副标题「保持专注，功德无量」
- 中间玻璃卡片包裹输入框
- 输入框：圆角矩形，浅灰背景，无边框
- 底部胶囊登录按钮

### 社区页
- 顶部 Tab 切换：「自律专区」|「鹿王专区」
- 帖子列表：每条是一个玻璃卡片
- 右下角浮动按钮：发布新帖

### 榜单页
- 顶部 Tab：「自律榜」|「功德榜」
- 排名列表：前三名用金银铜色标识

### 我的页
- 顶部用户信息卡片
- 数据统计
- 「数据分析」按钮
- 「退出登录」按钮

---

## 第4部分：SOS浮动球与木鱼

### SOS 浮动球
- 位置：右下角，距离底部导航栏上方 20px
- 外观：橙红色渐变（从 `#FFAA00` 到 `#FF5533`），直径 56px，白色「SOS」文字
- 效果：柔和阴影带橙色光晕 `box-shadow: 0 4px 20px rgba(255, 100, 50, 0.4)`

### 木鱼弹窗
- 背景：深色半透明遮罩 + 模糊
- 顶部：白色「累计功德: XXX」计数器
- 中央：大型木鱼图形（棕色调椭圆）
- 点击效果：缩放动画 + 飘出「功德+1」文字
- 底部提示：「点击敲击木鱼，平心静气」
- 右上角：关闭按钮

### 数据同步
- 本地累计点击次数
- 每 5 秒或关闭时批量上传
